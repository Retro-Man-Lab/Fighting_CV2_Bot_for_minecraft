# Fighting CV2 Bot for Minecraft

Бот для Minecraft который использует OpenCV для анализа экрана и автоматизации боя.

## Возможности

- Macros
- AutoHeal

---

# Установка

## Быстрый запуск

Скачайте `.exe` из папки `dist` и запустите: `main.exe`

---

## Запуск через Python

1. Установите Python 3.10+
2. Скачайте репозиторий
3. Установите зависимости
4. Запустите бота

```bash
python main.py
```

---

## Панель управления

```
         BOT PANEL
┌─────┬───────────┬───────┐
│ Key │ Function  │ State │
├─────┼───────────┼───────┤
│ 1   │ AutoClick │ ON    │
│ 2   │ AutoHeal  │ ON    │
│ Q   │ Exit      │       │
└─────┴───────────┴───────┘
```

---

## Управление
  * 1 — включить / выключить Macros
  * 2 — включить / выключить AutoHeal
  * Q — выйти из программы
